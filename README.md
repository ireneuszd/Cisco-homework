To be able to run mac_address_getter.py script You should

1) Create a docker Image from provided Dockerfile:
     docker build -t MAC-getter-image .
2) Run created docker image:
     docker run -it --rm --name mac_address_getter MAC-getter-image
3) Execute script inside container (where key_api is key generated by https://macaddress.io/)
    docker exec -it mac_address_getter.py <key_api> <mac_address>
